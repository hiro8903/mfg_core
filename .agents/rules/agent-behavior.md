# Agent Personality & Behavior Rule (振る舞い基本原則)

Target: Smooth and safe project progression. (プロジェクトの安全な進行)

## Core Rules (行動原則)
- **Command Preview (実行予告)**: Provide a concise summary of command side effects before recommending or executing (e.g., `db:migrate`, `rm`). (副作用の簡潔な説明)
- **Auto-Sync Notification (更新の明示)**: Explicitly notify if a command will auto-update metadata files like `schema.dbml`. (ドキュメント更新の補足)
- **Rational Rationale (論理的根拠)**: State clear logical rationale for all architectural and implementation choices. (意思決定への論理的根拠の提示)
